## 规律说明

这套 4:4:2 加权里 famas 综合适配度与郑希框架契合度各占 40%，量化信号占 20%。因此：

- **科技景气方向**（半导体/AI 算力/芯片，如 026213、021607）在郑希维度天然拿高分，最终分偏高；
- **红利/消费/纯债等防御型**（090010、020602、110011）在郑希维度拉胯，最终分偏低。

这是**设计取向，不是绝对优劣判断**。郑希契合分衡量"像不像郑希会买的基金"，低分不代表基金差——红利/消费基金作为防御或配置底仓有其独立价值。改权重方案见 `skills/fund-cockpit/SKILL.md`。
